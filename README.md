## 🚀 Key Features

### 1. Robust Component Capture (Deep Expansion)
Unlike standard tools that often crop absolute-positioned or overflow-hidden elements, our engine:
- Recursively expands all parent containers to `overflow: visible`.
- Dynamically resizes the browser viewport to match the element's natural `scrollWidth/Height`.
- Accounts for window offsets to ensure pixel-perfect clipping.

### 2. Performance: Browser Context Pooling
Maintains a single, long-lived Chromium process using a **Context Pool**. Concurrent tasks run in isolated tabs (targets), reducing startup overhead by ~80% during bulk operations.

### 3. Reliability & Wait Strategies
- **`document.readyState`**: Ensures the DOM is fully constructed.
- **`networkIdle`**: Grace period for AJAX and background assets.
- **`--wait-time`**: Customizable buffer for slow-rendering JS components.
- **Image Loading**: Specifically waits for all `<img>` tags to finish loading within the target element.

### 4. Advanced Dynamic Handling
- **Infinite Scroll**: Automated driver scrolls through the page to trigger lazy-load events.
- **Custom Injection**: Inject CSS or JS to hide banners, ads, or trigger specific UI states before capture.

## 🛠 Installation

### Install via Go (Recommended)
```bash
go install github.com/Abhay0thakor/html-to-image/cmd/html-to-image@latest
```

### Local Build
```bash
git clone https://github.com/Abhay0thakor/html-to-image.git
cd html-to-image
go build -o html-to-image ./cmd/html-to-image/main.go
```

## 📖 CLI Reference

| Flag | Default | Description |
| :--- | :--- | :--- |
| `-u, --url` | | Target URL(s) |
| `-s, --selector` | | CSS selector for specific component |
| `-o, --output` | `output.zip` | Filename or ZIP for bulk |
| `--format` | `png` | `png`, `jpeg`, or `pdf` |
| `--scale` | `1.0` | Device scale factor (DPI) |
| `--auto-scroll` | `false` | Enable infinite scroll driver |
| `--wait-time` | `0` | Extra seconds to wait for render |
| `--config` | | Path to YAML config file |
| `--inject-css` | | CSS to inject before capture |

## 📦 Using as a Library

```go
import (
    "context"
    "github.com/user/html-to-image/pkg/converter"
    "github.com/user/html-to-image/pkg/models"
)

func main() {
    conv := converter.NewConverter()
    conv.Start(context.Background())
    defer conv.Shutdown()

    img, _ := conv.Convert(context.Background(), models.ConversionConfig{
        Input: "https://www.toolsura.com",
        Selector: "header",
    })
}
```
